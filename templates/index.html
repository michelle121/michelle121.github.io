亮点

风格亮点

在保持原图像ID信息和纹理细节的同时，将真实照片转换为卡通风格的非真实感图像。卡通图像有清晰的边缘，平滑的色块和经过简化的纹理。风格更偏向写实，可有效保护个人隐私~

方法亮点

采用基于非成对数据的图像翻译（UNnaired Image Translation）方法进行图像转换，加入Face ID Loss,使用预训练的人脸识别模型提取照片和卡通化的ID特征，有效预防照片卡通化后容易丢失身份信息的问题！

数据亮点

仅需要少量数据就可以实现图像转换，本课题主要研究基于未配对数据的图像转换技术，未配对数据的形式广泛存在于我们日常生活中，不需要昂贵的人力标注。充分利用GAN的优势，来实现不成对图像的转换。

介绍

特征提取

特征提取部分包含堆叠的 Hourglass 模块、下采样模块和残差模块。其中，Hourglass 模块常用于姿态估计，它能够在特征提取的同时保持语义信息位置不变。该方法采用堆叠的 Hourglass 模块，渐进地将输入图像抽象成易于编码的形式。四个残差模块则用于编码特征并提取统计信息，用于后续的特征融合。

特征融合

近期论文U-GAT-IT提出了一种归一化方法——AdaLIN，能够自动调节Instance Norm和Layer Norm的比重，在结合注意力机制实现人像日漫风格转换。基于此提出了 Soft-AdaLIN（Soft Adaptive Layer-Instance Normalization）归一化方法。先将输入图像的编码特征统计信息和卡通特征统计信息融合，再以 AdaLIN 的方式反归一化解码特征，使卡通画结果更好地保持输入图像的语义内容。

特征重建

特征重建部分负责从编码特征生成对应的卡通图像。这部分采用与特征提取部分对称的结构，通过解码模块、上采样模块和 Hourglass 模块生成卡通画结果。

模型优化

照片卡通化后容易丢失身份信息。为了使输出结果体现人物信息，除了常规的 Cycle Loss 和 GAN Loss，引入 ID Loss：使用预训练的人脸识别模型来提取输入真人照和生成卡通画的 id 特征，使卡通画的 id 信息与输入照片尽可能相似。
